# Address Book in C

## Overview

Address Book is a C-based project that stores and manages contact details. It allows users to add, search, edit, delete, and view contacts such as Name, Mobile Number, and Email. The project uses file handling to load and save contact data persistently.

## Features

* Add new contacts
* Search contacts
* Edit existing contacts
* Delete contacts
* Display contact list
* Save contacts to file
* Load contacts from file

## Technologies Used

* C Programming
* File Handling
* Structures
* Functions
* Header Files

## Project Structure

* `main.c` – Main program execution
* `create.c` – Add contact
* `search.c` – Search functionality
* `edit.c` – Edit contact
* `delete.c` – Delete contact
* `list.c` – Display contacts
* `save.c` – Save contacts to file
* `load_file_data.c` – Load contacts from file

## How to Run

Compile and run the project using:

```bash
gcc *.c
./addressbook
```
